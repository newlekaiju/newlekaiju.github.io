安装个人博客 - hexo

node.js

命名为 http://.github.io

$ npm install -g hexo-cli
$ npm install hexo-server –save

安装hexo-deployer-git插件
$ npm install hexo-deployer-git –save

初始化博客

$ hexo init blog

测试hexo是否安装成功：

$ hexo new test
$ hexo g
$ hexo s

进入 http://localhost:4000 即可看到生成的网页

配置
_config.yml 文件
